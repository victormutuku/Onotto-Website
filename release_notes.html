Release Notes

Version 1.0.1.7

  • - Updated application to support Android SDK 33.
  • - Fixed issue with note height. Previously, when opening a note, the note height was not occupying the entire screen. This has been fixed.
  • - Fixed issue with notes not been saved when app is backgrounded and dismissed. This has been fixed.

Version 1.0.1.6

  • - Able to view the time and date when note was created. If the note was created in the last 24 hours, the time will be shown. If it was created later than 24 hours, the date will be shown.
  • - Notes are ordered by last modified date.
  • - Fixed issue with app exiting on clicking device back button when editing a note.

Version 1.0.1.5

  • - Able to save notes automatically on creating and updating.
  • - Fixed bug on updating note with empty title and content.
  • -

    Added alert dialog on updating note with empty title and content.

  • - Added alert dialog on deleting multiple notes.
  • - Modified font size of note content on viewing selected note.
  • - Removed empty settings page.

Version 1.0.1.4

  • - Fixed bug on updating notes.

Version 1.0.1.3

  • - Fixed UI issue: Overlapping of select boxes with Note Title
  • - Fixed App Display Name.
  • - Added functionality to delete button on selecting notes.
  • - Added functionality to appbar to count selected notes.
